# WorkLife DS

Design System do WorkLife Hub.

## Stack
- React 18 + TypeScript
- CSS Modules + tokens CSS 3 camadas
- Vite
- Lucide React

## Componentes
- **atoms** — Button, Badge, Avatar, Input, Checkbox, Toggle, Select, Radio, Spinner, Skeleton, ProgressBar, Tooltip, Tag, Card, ThemeToggle
- **molecules** — FormField, StatCard, Callout, Tabs, Pagination, SearchField, PasswordField, DatePicker, EmptyState, Dropdown, Accordion, Breadcrumb, Stepper
- **organisms** — Sidebar, Navbar, DataTable, Modal, Toast, CommandMenu, NotificationCenter, DailyCheckInStepper, MoodSelector, Drawer, PageHeader, FilterBar
- **product** — WellbeingScore, GoalCard, HabitTracker, ActivityFeed, FocusTimer, MoodHistory, InsightCard, OnboardingFlow, TeamDashboard

## Dev
```bash
npm install
npm run dev
```
